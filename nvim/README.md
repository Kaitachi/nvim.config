# Git Customizations
* `git config --global alias.graph "log --graph --pretty=oneline"`
* `git config --global alias.l "log --graph --pretty=format:'%Cred%h%Creset -%C(auto)%d%Creset %s %Cgreen(%cr)%Creset' --abbrev-commit --date=relative"`
